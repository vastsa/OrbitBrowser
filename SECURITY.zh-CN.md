# 安全策略

语言：[English](SECURITY.md) | 简体中文

Orbit Browser 是本地浏览器自动化工具，重点保护本机 Profile、Cookie、
代理凭据和任务产物。

## 支持版本

当前公开维护版本为 `0.1.x`。

## 报告安全问题

请不要在公开 Issue 中披露可利用细节。建议通过私有渠道联系维护者，并提供：

- 受影响版本或提交
- 复现步骤
- 影响范围
- 建议修复方向

## 安全边界

- 任务脚本运行在受控 JavaScript runtime 中，但它仍可以驱动本机浏览器。
- 代理账号、Cookie、Profile、截图和运行产物都应视为敏感数据。
- 不要把生产账号、生产代理或真实用户数据写入示例、测试或公开日志。
